# Eleraki Platform — Domain Vocabulary

هذا الملف هو **القاموس الرسمي** للغة الأعمال في المنصة.
كل Class سنكتبه لاحقاً يجب أن يكون موجوداً أولاً في هذا القاموس.

---

## المفاهيم الأساسية (Business Meta Model)

| المصطلح | المعنى |
|---------|--------|
| Enterprise | المؤسسة التي تمتلك وتشغل المنصة |
| Party | أي شخص أو جهة تتعامل مع المؤسسة (عميل، موظف، مورد، شريك...) |
| Person | شخص حقيقي (مندوب، موظف، عميل، مورد...) |
| Organization | جهة اعتبارية (شركة، مؤسسة، وزارة...) |
| Organization Unit | وحدة تنظيمية داخل المؤسسة |
| Position | منصب/دور وظيفي داخل Organization Unit |
| Resource | أصل أو مورد تمتلكه المؤسسة |
| Document | مستند يُنتج أو يُستهلك داخل المؤسسة |
| Process | سلسلة من الأنشطة تحقق هدفاً |
| Transaction | حدث يغير حالة النظام |
| Rule | قاعدة أعمال تحكم المؤسسة |

## Business Operating System (BOS)

### Foundation Engines
| Engine | المسؤولية |
|--------|-----------|
| Enterprise | إدارة المؤسسات والوحدات التنظيمية |
| Identity | إدارة الهويات والحسابات |
| Authorization | إدارة الصلاحيات والصلاحيات |
| Workflow | إدارة سير العمل والعمليات |
| Notification | إدارة الإشعارات والتنبيهات |
| Audit | تسجيل وتتبع جميع الأحداث |
| Configuration | إدارة إعدادات النظام |

### Business Engines
| Engine | المسؤولية |
|--------|-----------|
| Accounting | المحاسبة والدفاتر المالية |
| Inventory | إدارة المخزون |
| Purchasing | المشتريات |
| Sales | المبيعات |
| CRM | إدارة علاقات العملاء |
| HR | إدارة الموارد البشرية |
| Manufacturing | التصنيع |
| Projects | إدارة المشاريع |
| Assets | إدارة الأصول |

### Integration Engines
| Engine | المسؤولية |
|--------|-----------|
| Reporting | إنشاء التقارير |
| Analytics | التحليلات والبيانات |
| API Gateway | بوابة التكامل الخارجي |
| Messaging | نظام الرسائل |
| External Systems | التكامل مع أنظمة خارجية |

## قواعد التسمية

1. كل مصطلح يبدأ بحرف كبير.
2. لا نستخدم الاختصارات في أسماء الكلاسات.
3. كل مفهوم له كلاس واحد فقط.
4. إذا كان المصطلح موجوداً في هذا القاموس، نستخدمه كما هو.
5. إذا لم يكن موجوداً، نضيفه أولاً ثم نكتب الكود.

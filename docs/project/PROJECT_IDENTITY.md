WHY THIS PROJECT EXISTS
============================================================

The goal of Eleraki Platform is not to build another ERP system.

The goal is to build an engineering platform capable of understanding business
and transforming that understanding into reusable software.

ERP is only the first application that proves the platform.

============================================================

Architecture is designed before implementation.

Code is written only after understanding is complete.
